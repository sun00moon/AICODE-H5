一元二次方程单元 · 例题
练习 3/10

求解方程：x² + 5x + 6 = 0

请选择正确的解法和答案

A. 因式分解：(x+2)(x+3)=0 → x=2 或 x=3

B. 因式分解：(x+2)(x+3)=0 → x=-2 或 x=-3

正确答案！

C. 求根公式：x = [-5±√(25-24)]/2 → x=1 或 x=6

D. 配方法：(x+2.5)²=0.25 → x=3 或 x=2

解题步骤
  1. 观察方程 x² + 5x + 6 = 0，尝试因式分解
  2. 寻找两个数，它们的和为5，积为6 → 2和3
  3. 因此方程可分解为 (x+2)(x+3) = 0
  4. 令每个因式等于0，得到 x+2=0 或 x+3=0
  5. 解得 x=-2 或 x=-3
